# Media Assets for Web Showcase

Place these files in this folder. The site will show placeholders until they exist.

## Flow

1. **App explanation** (text panels) → 2. **App Demo** (phone) → 3. **Hardware** (Web3, specs) → 4. **Watch 3D** (render)

## App demo (floating phone)

| File | Used in |
|------|---------|
| `app-overview.mp4` | App Demo panel (floating 3D phone) |
| `app-overview-poster.jpg` | Poster before video loads |

## Images (hardware)

| File | Used in |
|------|---------|
| `smartwatch.png` | Hardware panel |
| `token-rewards.png` | Web3 panel |
| `watch-3d.png` | Watch 3D panel (floating 3D render) |

---

If a video or image fails to load (404), the panel will show a placeholder instead.
